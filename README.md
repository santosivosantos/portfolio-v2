# Portfolio V2

Personal portfolio built with Astro, hosted on GitHub Pages.

- **Domain:** [ivosantos.com](https://ivosantos.com)
- **Case studies:** BMW C Evolution, EMS Dashboard

## Tech Stack

- Astro (static site generation)
- GitHub Pages
- Inter

## Scripts

```bash
npm install
npm run dev    # local dev
npm run build  # build
npm run preview
```

## Deploy

GitHub Actions builds and deploys on every push to `main`.

## Custom Domain

The `public/CNAME` file contains `ivosantos.com`. Point your DNS A records to GitHub Pages IPs.
